# Worklog

![Go](https://github.com/PossibleLlama/worklog/workflows/Go/badge.svg)

Intented to be a quick productivity CLI tool, to track what you
have done each day.

## ~/.worklog/config.yml

Holds data provided to the worklog app.

``` yml
author: "Alice"
default:
  recordLocation: "~/my/new/location"
  duration: 15
```
